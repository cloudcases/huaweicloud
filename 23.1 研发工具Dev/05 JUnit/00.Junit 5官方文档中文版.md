# Junit 5官方文档中文版

## 介绍

JUnit 5是JUnit的下一代。目标是为JVM上的开发人员端测试创建一个最新的基础。这包括专注于Java 8及更高版本，以及启用许多不同风格的测试。

JUnit 5团队于2017年9月10日发布了第一个[GA版本 5.0.0][ga]。

这里是Junit5官方文档的中文翻译版，由[doczh.cn](http://doczh.cn/)组织翻译并更新维护。
